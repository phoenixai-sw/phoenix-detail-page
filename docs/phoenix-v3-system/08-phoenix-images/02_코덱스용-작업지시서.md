# phoenix images 코덱스용 작업지시서

## 1. 작업 목표

`phoenix images`를 Design Bot이 컨트롤할 수 있는 제품 이미지/카드뉴스/썸네일 생성 웹으로 구축한다.

## 2. 필수 화면

1. 이미지 용도 선택
2. 제품명/브랜드명 입력
3. 참고 이미지 업로드
4. 스타일/비율/파일 형식 선택
5. 생성 버튼
6. 결과 미리보기
7. PNG/JPEG 다운로드
8. ZIP 다운로드

## 3. 자동화 식별자

```text
data-testid="phoenix-images-purpose"
data-testid="phoenix-images-brand"
data-testid="phoenix-images-product"
data-testid="phoenix-images-upload"
data-testid="phoenix-images-style"
data-testid="phoenix-images-aspect-ratio"
data-testid="phoenix-images-format"
data-testid="phoenix-images-generate"
data-testid="phoenix-images-preview"
data-testid="phoenix-images-download"
data-testid="phoenix-images-download-zip"
```

## 4. API 후보

```text
src/app/api/images/generate/route.ts
src/app/api/images/export/route.ts
src/app/api/images/export-zip/route.ts
```

## 5. 성공 기준

1. 제품 목업을 생성할 수 있다.
2. 썸네일을 생성할 수 있다.
3. 카드뉴스 이미지를 생성할 수 있다.
4. PNG/JPEG 다운로드가 가능하다.
5. Design Bot이 자동 조작할 수 있다.

## 6. 금지 사항

1. 저작권 문제가 있는 인물/브랜드 이미지를 무단 생성하지 않는다.
2. 유저가 업로드한 원본 이미지를 공개 경로에 노출하지 않는다.
3. 실패한 이미지를 성공 결과로 표시하지 않는다.
