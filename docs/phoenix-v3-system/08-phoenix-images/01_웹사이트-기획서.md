# phoenix images 웹사이트 기획서

## 1. 정의

`phoenix images`는 제품 목업, 제품컷, 카드뉴스, 썸네일을 생성하는 이미지 제작 agent web이다.

## 2. 컨트롤 봇

```text
Design Bot
```

Design Bot은 이미지 콘셉트, 브랜드 톤, 레이아웃, 썸네일 구성을 판단하므로 `phoenix images`의 주 컨트롤 봇으로 적합하다.

## 3. 주요 유저

1. 쇼핑몰 운영자
2. 강의/콘텐츠 제작자
3. 광고 소재 제작자
4. 상세페이지 제작자
5. 브랜드 마케터

## 4. 핵심 기능

1. 제품 목업 생성
2. 제품컷 생성
3. 카드뉴스 이미지 생성
4. 썸네일 생성
5. 배경 제거/교체
6. 브랜드 컬러 반영
7. PNG/JPEG 다운로드
8. ZIP 묶음 다운로드
9. V3 이후 Design Bot 자동 조작

## 5. 결과물

1. PNG
2. JPEG
3. 카드뉴스 세트
4. 썸네일 이미지
5. 제품 목업 이미지

## 6. V3 흐름

```text
마스터 → Genesis Bot에게 이미지 소재 요청
→ Genesis Bot이 Design Bot에게 task 전달
→ Design Bot이 phoenix images 접속
→ 제품/브랜드/용도 입력
→ 이미지 생성
→ PNG/JPEG 다운로드
→ Telegram 또는 Cloud로 전달
```

## 7. 구축 우선순위

`phoenix images`는 `phoenix pages`와 가장 가까운 확장 기능이다.

우선순위:

1. 이미지 생성 기본 폼
2. 용도별 프리셋
3. PNG/JPEG 다운로드
4. ZIP 다운로드
5. Design Bot 자동화 식별자
