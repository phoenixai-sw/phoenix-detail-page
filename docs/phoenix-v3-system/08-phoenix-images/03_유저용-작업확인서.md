# phoenix images 유저용 작업확인서

## 1. 목적

`phoenix images`가 제품 목업, 제품컷, 카드뉴스, 썸네일 생성 웹으로 정상 작동하는지 확인한다.

## 2. 기본 생성 확인

1. 사이트에 접속한다.
2. 이미지 용도를 선택한다.
3. 제품명과 브랜드명을 입력한다.
4. 참고 이미지를 업로드한다.
5. 스타일과 비율을 선택한다.
6. 생성 버튼을 누른다.

성공 기준:

```text
결과 이미지가 화면에 보인다.
```

## 3. 다운로드 확인

1. PNG 다운로드를 확인한다.
2. JPEG 다운로드를 확인한다.
3. 여러 이미지가 있을 경우 ZIP 다운로드를 확인한다.

## 4. Design Bot 자동화 확인

1. Telegram에서 Design Bot에게 썸네일 생성을 요청한다.
2. Design Bot이 필요한 정보를 질문한다.
3. Design Bot이 phoenix images를 조작한다.
4. 결과 이미지를 전달한다.

## 5. 실패 확인

1. 이미지 생성 실패 시 원인이 표시되는가?
2. 다운로드 실패 시 다시 시도할 수 있는가?
3. 파일 용량이 클 때 안내가 나오는가?
