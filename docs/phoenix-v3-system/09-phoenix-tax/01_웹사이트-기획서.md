# phoenix tax 웹사이트 기획서

## 1. 정의

`phoenix tax`는 양도소득세 관련 정보, 해당 세무사 사무실 정보, 세무사님 소개, agent 채팅상담을 제공하는 세무 상담 agent web이다.

## 2. 컨트롤 봇

```text
Power Bot
```

Power Bot은 리서치, 근거 확인, 상담 요약, 문서 생성에 적합하다.

## 3. 핵심 기능

1. 양도소득세 기본 정보 안내
2. 세무사 사무실 소개
3. 세무사님 프로필 소개
4. agent 채팅상담
5. 상담 요약 생성
6. PDF 안내서 생성
7. Word 문서 생성
8. text 상담 기록 생성
9. 상담 예약 또는 문의 연결

## 4. 결과물

1. agent 채팅상담
2. PDF
3. Word
4. text

## 5. 주의 사항

세무 상담은 고위험 정보다.

따라서 문서와 상담 화면에는 아래 원칙을 반영한다.

1. 일반 정보와 개별 세무 자문을 구분한다.
2. 최종 판단은 세무사 상담이 필요하다고 안내한다.
3. 최신 세법 확인 필요성을 표시한다.
4. 고객 개인정보를 안전하게 처리한다.

## 6. V3 흐름

```text
마스터 또는 유저가 세무 상담 요청
→ Power Bot이 phoenix tax 접속
→ 질문 수집
→ 상담 답변 생성
→ 요약 문서 생성
→ PDF/Word/text 전달
```
