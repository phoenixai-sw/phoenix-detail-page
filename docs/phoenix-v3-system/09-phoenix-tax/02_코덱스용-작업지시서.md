# phoenix tax 코덱스용 작업지시서

## 1. 작업 목표

`phoenix tax`를 Power Bot이 컨트롤할 수 있는 세무 정보/채팅 상담/문서 생성 웹으로 구축한다.

## 2. 필수 화면

1. 세무사 사무실 소개
2. 세무사님 프로필
3. 양도소득세 기본 안내
4. 상담 채팅창
5. 상담 요약
6. PDF/Word/text 다운로드
7. 상담 예약/문의 버튼

## 3. 자동화 식별자

```text
data-testid="phoenix-tax-chat-input"
data-testid="phoenix-tax-chat-send"
data-testid="phoenix-tax-chat-history"
data-testid="phoenix-tax-summary"
data-testid="phoenix-tax-download-pdf"
data-testid="phoenix-tax-download-docx"
data-testid="phoenix-tax-download-text"
data-testid="phoenix-tax-contact"
```

## 4. API 후보

```text
src/app/api/tax/chat/route.ts
src/app/api/tax/summary/route.ts
src/app/api/tax/export-pdf/route.ts
src/app/api/tax/export-docx/route.ts
src/app/api/tax/export-text/route.ts
```

## 5. 성공 기준

1. 유저가 세무 질문을 입력할 수 있다.
2. agent 채팅상담이 가능하다.
3. 상담 요약을 생성할 수 있다.
4. PDF/Word/text 다운로드가 가능하다.
5. Power Bot이 자동 조작할 수 있다.

## 6. 금지 사항

1. 확정 세무 자문처럼 표현하지 않는다.
2. 최신 세법 확인 없이 단정하지 않는다.
3. 개인정보를 로그에 노출하지 않는다.
4. 세무사 확인이 필요한 사안을 AI 단독 결론으로 표시하지 않는다.
