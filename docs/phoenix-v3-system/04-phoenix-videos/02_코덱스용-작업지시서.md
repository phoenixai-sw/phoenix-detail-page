# phoenix videos 코덱스용 작업지시서

## 1. 작업 목표

`phoenix videos`를 Video Bot이 컨트롤할 수 있는 영상 생성 웹으로 구축한다.

## 2. 핵심 화면

1. 영상 목적 선택
2. 참고자료 업로드
3. 장면 구성 편집
4. 영상 생성 상태
5. 결과 미리보기
6. 다운로드

## 3. 자동화용 식별자

```text
data-testid="phoenix-videos-purpose"
data-testid="phoenix-videos-upload"
data-testid="phoenix-videos-duration"
data-testid="phoenix-videos-scene-count"
data-testid="phoenix-videos-style"
data-testid="phoenix-videos-generate"
data-testid="phoenix-videos-status"
data-testid="phoenix-videos-preview"
data-testid="phoenix-videos-download-mp4"
```

## 4. API 후보

```text
src/app/api/videos/storyboard/route.ts
src/app/api/videos/generate/route.ts
src/app/api/videos/status/route.ts
src/app/api/videos/download/route.ts
```

## 5. 작업 상태

영상은 오래 걸리므로 상태 API가 필수다.

```text
queued
storyboarding
generating_scene
rendering
completed
failed
```

## 6. 금지 사항

1. 긴 영상을 첫 버전 목표로 잡지 않는다.
2. 모든 장면을 한 번에 실패하면 전체 실패로 처리하지 않는다.
3. 파일 크기 제한 없이 Telegram 전송을 시도하지 않는다.
4. 크레딧 차감 없이 고비용 생성을 실행하지 않는다.

## 7. 1차 성공 기준

1. 15초 이하 영상 생성 흐름
2. 스토리보드 생성
3. MP4 결과 다운로드
4. Video Bot 자동 조작 가능
