# phoenix videos 웹사이트 기획서

## 1. 정의

`phoenix videos`는 영상 생성 웹이다.

숏폼, 광고 영상, 강의 예고 영상, 제품 소개 영상을 생성하거나 기획하는 웹사이트로 구축한다.

## 2. 주 컨트롤 봇

```text
Video Bot
```

Video Bot은 영상 콘셉트, 스토리보드, 장면 프롬프트, 영상 생성 상태 확인을 담당한다.

## 3. 핵심 기능

1. 영상 목적 선택
2. 영상 길이 선택
3. 장면 수 선택
4. 스타일 선택
5. 참고 이미지 업로드
6. 장면별 프롬프트 생성
7. 영상 생성
8. 썸네일 생성
9. MP4 다운로드
10. Telegram 결과 전달

## 4. 추천 메뉴

1. 숏폼 영상
2. 광고 영상
3. 제품 소개 영상
4. 강의 홍보 영상
5. 브랜드 무드 영상
6. 스토리보드만 만들기

## 5. 결과물

1. MP4
2. 썸네일 이미지
3. 장면별 프롬프트
4. 스토리보드 PDF
5. 생성 로그

## 6. 주의점

영상은 이미지보다 비용, 시간, 실패율, 파일 크기 부담이 크다.

따라서 `phoenix videos`는 pages/slides보다 늦게 구축하는 것이 안전하다.
