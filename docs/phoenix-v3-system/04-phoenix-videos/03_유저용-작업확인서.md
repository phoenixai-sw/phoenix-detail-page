# phoenix videos 유저용 작업확인서

## 1. 목적

`phoenix videos`가 영상 생성 웹으로 정상 작동하는지 확인한다.

## 2. 스토리보드 생성 확인

1. 영상 목적을 선택한다.
2. 영상 길이를 선택한다.
3. 장면 수를 선택한다.
4. 생성 버튼을 누른다.

성공 기준:

```text
장면별 스토리보드가 생성된다.
```

## 3. 영상 생성 확인

1. 스토리보드를 확인한다.
2. 영상 생성을 실행한다.
3. 진행 상태가 표시되는지 확인한다.

## 4. 다운로드 확인

1. 영상 생성 완료 후 MP4 다운로드를 누른다.
2. 영상 파일이 재생되는지 확인한다.

## 5. Video Bot 자동화 확인

1. Telegram에서 Video Bot에게 영상 생성을 요청한다.
2. Video Bot이 필요한 조건을 질문한다.
3. Video Bot이 phoenix videos를 조작한다.
4. 결과 MP4 또는 링크를 전달받는다.
