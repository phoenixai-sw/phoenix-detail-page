# phoenix books 코덱스용 작업지시서

## 1. 작업 목표

`phoenix books`를 Writer Bot이 컨트롤할 수 있는 강의교안/SNS 카피라이팅 생성 웹으로 구축한다.

## 2. 핵심 화면

1. 콘텐츠 유형 선택
2. 주제 입력
3. 대상 독자/수강생 입력
4. 톤 선택
5. 분량 선택
6. 목차/구성 미리보기
7. 본문 생성
8. DOCX/PDF 다운로드

## 3. 자동화용 식별자

```text
data-testid="phoenix-books-content-type"
data-testid="phoenix-books-topic"
data-testid="phoenix-books-audience"
data-testid="phoenix-books-tone"
data-testid="phoenix-books-length"
data-testid="phoenix-books-generate-outline"
data-testid="phoenix-books-generate-body"
data-testid="phoenix-books-preview"
data-testid="phoenix-books-download-docx"
data-testid="phoenix-books-download-pdf"
```

## 4. API 후보

```text
src/app/api/books/outline/route.ts
src/app/api/books/generate/route.ts
src/app/api/books/export-docx/route.ts
src/app/api/books/export-pdf/route.ts
```

## 5. 데이터 구조 후보

```text
book_projects
book_sections
book_copy_sets
book_exports
```

## 6. 문서 생성 기준

1. 강의교안은 목차, 학습목표, 본문, 실습, 마무리로 구성한다.
2. SNS 카피는 플랫폼별로 길이와 톤을 조정한다.
3. DOCX는 편집 가능한 구조여야 한다.
4. PDF는 제출/공유용으로 깔끔하게 렌더링한다.
5. 출처가 필요한 내용은 근거 표시 영역을 둔다.

## 7. 금지 사항

1. 처음부터 장편 도서 집필 전체를 목표로 잡지 않는다.
2. 검증되지 않은 사실을 확정적으로 쓰지 않는다.
3. 민감한 개인정보를 문서에 자동 삽입하지 않는다.
4. Writer Bot이 생성한 문안을 검토 없이 최종본으로 표시하지 않는다.

## 8. 1차 성공 기준

1. 강의교안 목차 생성
2. 강의교안 본문 생성
3. SNS 카피 묶음 생성
4. DOCX 다운로드
5. Writer Bot 자동 조작 가능
