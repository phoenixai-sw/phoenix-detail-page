# phoenix books 유저용 작업확인서

## 1. 목적

`phoenix books`가 강의교안/SNS 카피라이팅 생성 웹으로 정상 작동하는지 확인한다.

## 2. 강의교안 생성 확인

1. 콘텐츠 유형에서 `강의교안`을 선택한다.
2. 강의 주제를 입력한다.
3. 대상 수강생을 입력한다.
4. 차시 또는 분량을 선택한다.
5. 목차 생성을 누른다.

성공 기준:

```text
강의교안 목차가 생성된다.
```

## 3. 본문 생성 확인

1. 목차를 확인한다.
2. 본문 생성을 누른다.
3. 학습목표, 본문, 실습 과제, 마무리가 생성되는지 확인한다.

## 4. SNS 카피 생성 확인

1. 콘텐츠 유형에서 `SNS 카피`를 선택한다.
2. 상품/강의 정보를 입력한다.
3. 플랫폼을 선택한다.
4. 카피 생성을 누른다.

성공 기준:

```text
여러 개의 SNS 카피 문안이 생성된다.
```

## 5. 다운로드 확인

1. DOCX 다운로드를 누른다.
2. 문서가 열리는지 확인한다.
3. PDF 다운로드가 있다면 PDF도 확인한다.

## 6. Writer Bot 자동화 확인

1. Telegram에서 Writer Bot에게 교안 생성을 요청한다.
2. Writer Bot이 주제/대상/분량을 질문한다.
3. Writer Bot이 phoenix books를 조작한다.
4. DOCX 또는 PDF 결과를 전달받는다.
