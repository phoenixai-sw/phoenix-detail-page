# phoenix books 웹사이트 기획서

## 1. 정의

`phoenix books`는 강의교안과 SNS 카피라이팅 생성 웹이다.

책 전체 집필 시스템이 아니라, 수업 자료와 콘텐츠 문안을 빠르게 생성하고 문서 파일로 정리하는 웹사이트로 시작한다.

## 2. 주 컨트롤 봇

```text
Writer Bot
```

Writer Bot은 원고, 카피라이팅, 교안, 문체 정리, 편집 흐름에 적합하다.

## 3. 주요 유저

1. 강사
2. 코치/컨설턴트
3. SNS 운영자
4. 마케터
5. 수강생 유저
6. 교재/워크북 제작자

## 4. 핵심 기능

1. 강의 주제 입력
2. 대상 수강생 입력
3. 수업 시간/차시 선택
4. 교안 목차 생성
5. 강의교안 본문 생성
6. 실습 과제 생성
7. SNS 카피 생성
8. 카드뉴스 문안 생성
9. DOCX 다운로드
10. PDF 다운로드
11. V3 이후 Writer Bot 자동 조작

## 5. 추천 메뉴

1. 강의교안 생성
2. 워크북 초안 생성
3. SNS 홍보글 생성
4. 카드뉴스 문안 생성
5. 강의 소개문 생성
6. 이메일/문자 안내문 생성

## 6. 결과물

1. DOCX 강의교안
2. PDF 강의교안
3. SNS 카피 묶음
4. 카드뉴스 문안
5. 실습 과제 목록

## 7. V3 자동화 흐름

```text
Genesis Bot 또는 마스터
→ Writer Bot에게 교안/카피 생성 요청
→ Writer Bot이 phoenix books 접속
→ 주제/대상/차시/톤 입력
→ 교안 또는 카피 생성
→ DOCX/PDF 다운로드
→ Telegram 또는 Cloud로 전달
```

## 8. 구축 우선순위

`phoenix books`는 `phoenix webs`와 함께 콘텐츠 생산 확장 축으로 구축한다.

다만 완성형 책 집필보다 강의교안과 SNS 카피라이팅을 먼저 목표로 잡는다.
