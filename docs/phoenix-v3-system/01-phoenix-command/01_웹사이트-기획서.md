# phoenix command 웹사이트 기획서

## 1. 정의

`phoenix command`는 Phoenix V3 시스템의 최상위 지휘/상태 관리 웹이다.

Genesis Bot이 전체 작업을 지휘하고, 마스터가 여러 agent web의 작업 상태와 결과물을 한 곳에서 보는 웹사이트다.

## 2. 주 컨트롤 봇

```text
Genesis Bot
```

Genesis Bot은 작업을 분석하고, 전문 봇에게 나누고, 최종 결과를 보고한다.

## 6. V3 대표 흐름

```text
마스터: 이 상품으로 상세페이지, 제품컷, 랜딩페이지, PPT, SNS 카피까지 만들어줘
→ Genesis Bot이 작업을 task로 분리
→ Design Bot: phoenix pages
→ Design Bot: phoenix images
→ Design Bot: phoenix webs
→ Design Bot: phoenix slides
→ Writer Bot: phoenix books
→ phoenix command에서 상태 표시
→ 결과물이 한 작업 묶음으로 저장
```

상담형 흐름:

```text
마스터: 이 고객사의 AI 검색 노출 현황을 분석하고 상담 보고서를 만들어줘
→ Genesis Bot이 Power Bot에게 task 전달
→ Power Bot: phoenix marketing
→ 대시보드 업데이트
→ 상담 요약과 보고서 생성
→ phoenix command에서 결과 확인
```
