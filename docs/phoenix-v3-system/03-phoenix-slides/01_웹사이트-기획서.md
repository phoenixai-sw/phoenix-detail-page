# phoenix slides 웹사이트 기획서

## 1. 정의

`phoenix slides`는 PPT 생성 웹이다.

강의자료, 제안서, 세미나 자료, 보고 발표자료를 자동 생성하는 웹사이트로 구축한다.

## 2. 주 컨트롤 봇

```text
Design Bot
```

Design Bot은 레이아웃, 색상, 브랜드 비주얼, 슬라이드 구성을 담당한다.

## 3. 주요 유저

1. 강사
2. 컨설턴트
3. 마케터
4. 스타트업 대표
5. 수강생 유저

## 4. 핵심 기능

1. 주제 입력
2. 참고자료 업로드
3. 슬라이드 장수 선택
4. 발표 톤 선택
5. 브랜드 컬러 선택
6. PPT 구조 생성
7. 슬라이드별 이미지/카피 생성
8. PPTX 다운로드
9. PDF 다운로드
10. 슬라이드 이미지 다운로드
11. V3 이후 Design Bot 자동 조작

## 5. 결과물

1. PPTX
2. PDF
3. 슬라이드별 PNG
4. 발표자 노트
5. 강의 흐름표

## 6. 추천 메뉴

1. 강의자료
2. 제안서
3. 세미나 발표
4. 상품 소개서
5. 투자/IR 초안
6. 카드뉴스형 슬라이드

## 7. V3 자동화 흐름

```text
Genesis Bot 또는 마스터
→ Design Bot에게 PPT 생성 요청
→ Design Bot이 phoenix slides 접속
→ 자료 업로드
→ 슬라이드 유형/장수/톤 선택
→ PPT 생성
→ PPTX/PDF 다운로드
→ Telegram으로 전달
```
