# phoenix slides 유저용 작업확인서

## 1. 목적

`phoenix slides`가 PPT 생성 웹으로 정상 작동하는지 확인한다.

## 2. 기본 생성 확인

1. 주제를 입력한다.
2. 슬라이드 장수를 선택한다.
3. 발표 톤을 선택한다.
4. 생성 버튼을 누른다.

성공 기준:

```text
슬라이드 목차와 미리보기가 생성된다.
```

## 3. 참고자료 업로드 확인

1. PDF 또는 이미지 자료를 업로드한다.
2. 자료 기반으로 슬라이드 내용이 반영되는지 확인한다.

## 4. 다운로드 확인

1. PPTX 다운로드 버튼을 누른다.
2. 파일이 열리는지 확인한다.
3. PDF 다운로드가 있다면 PDF도 확인한다.

## 5. Design Bot 자동화 확인

1. Telegram에서 Design Bot에게 PPT 생성을 요청한다.
2. Design Bot이 주제/장수/톤을 확인한다.
3. Design Bot이 phoenix slides를 조작한다.
4. 결과 PPTX를 전달받는다.
