# phoenix webs 유저용 작업확인서

## 1. 목적

`phoenix webs`가 웹사이트/랜딩페이지 HTML 생성 웹으로 정상 작동하는지 확인한다.

## 2. 기본 생성 확인

1. 웹사이트 목적을 입력한다.
2. 브랜드명 또는 서비스명을 입력한다.
3. 랜딩페이지 유형을 선택한다.
4. 섹션 구성을 선택한다.
5. CTA 문구를 입력한다.
6. 생성 버튼을 누른다.

성공 기준:

```text
랜딩페이지 미리보기가 생성된다.
```

## 3. 이미지 업로드 확인

1. 로고 또는 참고 이미지를 업로드한다.
2. 생성된 랜딩페이지에 이미지가 반영되는지 확인한다.

## 4. 다운로드 확인

1. ZIP 다운로드 버튼을 누른다.
2. ZIP 파일 안에 `index.html`, `style.css`, `assets`가 있는지 확인한다.
3. `index.html`을 브라우저에서 열어본다.

## 5. Design Bot 자동화 확인

1. Telegram에서 Design Bot에게 랜딩페이지 생성을 요청한다.
2. Design Bot이 필요한 정보를 질문한다.
3. Design Bot이 phoenix webs를 조작한다.
4. 결과 ZIP을 전달받는다.
