# phoenix marketing 웹사이트 기획서

## 1. 정의

`phoenix marketing`은 고객사별 AI 검색 순위, AI 플랫폼별 인용 정보, 예상 AI 검색 질문, 생성 데이터를 관리하는 대시보드와 agent 채팅상담을 제공하는 마케팅 agent web이다.

## 2. 컨트롤 봇

```text
Power Bot
```

Power Bot은 리서치, 순위 데이터 정리, 질문 추론, 보고서 생성, 상담 흐름에 적합하다.

## 3. 핵심 기능

1. 고객사 관리
2. AI 플랫폼별 상위 순위 기록
3. AI 검색 인용 정보 관리
4. 예상 AI 검색 질문 추론
5. 생성 데이터 관리 대시보드
6. agent 채팅상담
7. PDF/Word/text 보고서 생성
8. 변화 추적
9. 개선 제안 생성

## 4. 결과물

1. 고객사별 AI 플랫폼 순위 대시보드
2. AI 검색 인용 정보 목록
3. 예상 AI 검색 질문 목록
4. agent 채팅상담
5. PDF
6. Word
7. text

## 5. 주요 화면

1. 고객사 목록
2. 고객사 상세 대시보드
3. AI 플랫폼별 순위 테이블
4. 인용 정보 관리
5. 예상 질문 관리
6. 상담 채팅
7. 보고서 다운로드

## 6. V3 흐름

```text
마스터 또는 고객사가 AI 검색 현황 분석 요청
→ Power Bot이 phoenix marketing 접속
→ 고객사/플랫폼/키워드 조건 입력
→ 순위와 인용 정보 정리
→ 예상 질문 추론
→ 대시보드 업데이트
→ 상담 및 보고서 생성
```
