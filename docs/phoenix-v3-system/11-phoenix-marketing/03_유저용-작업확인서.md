# phoenix marketing 유저용 작업확인서

## 1. 목적

`phoenix marketing`이 고객사별 AI 검색 순위, 인용 정보, 예상 질문, 대시보드, 채팅상담, 보고서 다운로드 기능을 제공하는지 확인한다.

## 2. 고객사 관리 확인

1. 고객사를 등록한다.
2. 고객사 상세 화면에 접속한다.
3. 고객사 정보를 수정한다.

## 3. 대시보드 확인

1. AI 플랫폼별 순위 항목이 보이는지 확인한다.
2. 인용 정보 목록이 보이는지 확인한다.
3. 예상 AI 검색 질문 목록이 보이는지 확인한다.
4. 최근 업데이트 시간이 보이는지 확인한다.

## 4. 채팅상담 확인

1. 고객사 관련 질문을 입력한다.
2. 답변이 생성되는지 확인한다.
3. 답변이 대시보드 데이터와 연결되는지 확인한다.

## 5. 보고서 다운로드 확인

1. PDF 다운로드를 확인한다.
2. Word 다운로드를 확인한다.
3. text 다운로드를 확인한다.

## 6. Power Bot 자동화 확인

1. Telegram에서 Power Bot에게 고객사 AI 검색 현황 분석을 요청한다.
2. Power Bot이 고객사/플랫폼/키워드를 질문한다.
3. Power Bot이 phoenix marketing을 조작한다.
4. 대시보드 업데이트 결과와 보고서를 전달한다.
