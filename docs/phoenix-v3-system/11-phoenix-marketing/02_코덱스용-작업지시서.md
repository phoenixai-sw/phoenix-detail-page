# phoenix marketing 코덱스용 작업지시서

## 1. 작업 목표

`phoenix marketing`을 Power Bot이 컨트롤할 수 있는 AI 검색 순위/인용 정보/예상 질문 관리 대시보드 및 채팅 상담 웹으로 구축한다.

## 2. 필수 화면

1. 고객사 목록
2. 고객사 등록/수정
3. 고객사 상세 대시보드
4. AI 플랫폼별 순위 관리
5. 인용 정보 관리
6. 예상 AI 검색 질문 관리
7. 상담 채팅창
8. PDF/Word/text 보고서 다운로드

## 3. 자동화 식별자

```text
data-testid="phoenix-marketing-client-list"
data-testid="phoenix-marketing-client-create"
data-testid="phoenix-marketing-client-detail"
data-testid="phoenix-marketing-platform-rank"
data-testid="phoenix-marketing-citation-list"
data-testid="phoenix-marketing-question-list"
data-testid="phoenix-marketing-chat-input"

data-testid="phoenix-marketing-download-pdf"
data-testid="phoenix-marketing-download-docx"
data-testid="phoenix-marketing-download-text"
```


## 6. 성공 기준

1. 고객사를 등록하고 관리할 수 있다.
2. AI 플랫폼별 순위를 기록할 수 있다.
3. 인용 정보를 관리할 수 있다.
4. 예상 AI 검색 질문을 생성하고 저장할 수 있다.
5. agent 채팅상담이 가능하다.
6. PDF/Word/text 보고서 다운로드가 가능하다.
7. Power Bot이 자동 조작할 수 있다.

## 7. 금지 사항

1. 실제 검색 검증 없이 순위를 확정값처럼 표시하지 않는다.
2. 고객사 내부 정보를 공개 로그에 노출하지 않는다.
3. AI 플랫폼별 결과를 출처 없이 단정하지 않는다.
4. 대시보드 데이터와 보고서 데이터가 서로 어긋나게 만들지 않는다.
