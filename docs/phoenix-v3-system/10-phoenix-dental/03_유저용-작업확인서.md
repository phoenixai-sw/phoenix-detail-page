# phoenix dental 유저용 작업확인서

## 1. 목적

`phoenix dental`이 치아관리 정보, 치과병원 소개, agent 채팅상담, 문서 다운로드 기능을 제공하는지 확인한다.

## 2. 정보 화면 확인

1. 치과병원 정보가 보이는지 확인한다.
2. 치과원장님 프로필이 보이는지 확인한다.
3. 치아관리 기본 안내가 보이는지 확인한다.

## 3. 채팅상담 확인

1. 상담 질문을 입력한다.
2. 답변이 생성되는지 확인한다.
3. 진단이 아니라 일반 정보 안내로 표현되는지 확인한다.
4. 병원 진료 필요 안내가 보이는지 확인한다.

## 4. 문서 다운로드 확인

1. 상담 요약을 생성한다.
2. PDF 다운로드를 확인한다.
3. Word 다운로드를 확인한다.
4. text 다운로드를 확인한다.

## 5. Power Bot 자동화 확인

1. Telegram에서 Power Bot에게 치아관리 상담 준비를 요청한다.
2. Power Bot이 필요한 정보를 질문한다.
3. Power Bot이 phoenix dental을 조작한다.
4. 상담 요약과 문서 링크를 전달한다.
