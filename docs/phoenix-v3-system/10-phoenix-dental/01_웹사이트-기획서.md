# phoenix dental 웹사이트 기획서

## 1. 정의

`phoenix dental`은 치아관리 정보, 해당 치과병원 정보, 치과원장님 소개, agent 채팅상담을 제공하는 치과 상담 agent web이다.

## 2. 컨트롤 봇

```text
Power Bot
```

Power Bot은 정보 정리, 상담 요약, 문서 생성, 안전 표현 관리에 적합하다.

## 3. 핵심 기능

1. 치아관리 정보 안내
2. 치과병원 소개
3. 치과원장님 프로필 소개
4. agent 채팅상담
5. 상담 요약 생성
6. PDF 안내서 생성
7. Word 문서 생성
8. text 상담 기록 생성
9. 예약 또는 문의 연결

## 4. 결과물

1. agent 채팅상담
2. PDF
3. Word
4. text

## 5. 주의 사항

치과/의료 정보는 고위험 정보다.

설계 원칙:

1. 일반 정보와 진단을 구분한다.
2. 최종 진단은 치과의사의 대면 진료가 필요하다고 안내한다.
3. 응급 증상은 즉시 병원 방문을 안내한다.
4. 개인정보와 건강정보를 안전하게 처리한다.

## 6. V3 흐름

```text
유저가 치아관리 또는 진료 상담 요청
→ Power Bot이 phoenix dental 접속
→ 질문 수집
→ 일반 정보 중심 상담 답변 생성
→ 요약 문서 생성
→ PDF/Word/text 전달
```
