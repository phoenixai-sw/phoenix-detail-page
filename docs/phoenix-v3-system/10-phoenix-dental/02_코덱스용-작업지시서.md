# phoenix dental 코덱스용 작업지시서

## 1. 작업 목표

`phoenix dental`을 Power Bot이 컨트롤할 수 있는 치아관리 정보/채팅 상담/문서 생성 웹으로 구축한다.

## 2. 필수 화면

1. 치과병원 소개
2. 치과원장님 프로필
3. 치아관리 기본 안내
4. 상담 채팅창
5. 상담 요약
6. PDF/Word/text 다운로드
7. 예약/문의 버튼

## 3. 자동화 식별자

```text
data-testid="phoenix-dental-chat-input"
data-testid="phoenix-dental-chat-send"
data-testid="phoenix-dental-chat-history"
data-testid="phoenix-dental-summary"
data-testid="phoenix-dental-download-pdf"
data-testid="phoenix-dental-download-docx"
data-testid="phoenix-dental-download-text"
data-testid="phoenix-dental-reservation"
```

## 4. API 후보

```text
src/app/api/dental/chat/route.ts
src/app/api/dental/summary/route.ts
src/app/api/dental/export-pdf/route.ts
src/app/api/dental/export-docx/route.ts
src/app/api/dental/export-text/route.ts
```

## 5. 성공 기준

1. 유저가 치아관리 질문을 입력할 수 있다.
2. agent 채팅상담이 가능하다.
3. 상담 요약을 생성할 수 있다.
4. PDF/Word/text 다운로드가 가능하다.
5. Power Bot이 자동 조작할 수 있다.

## 6. 금지 사항

1. 진단/처방처럼 단정하지 않는다.
2. 응급 증상을 일반 안내로만 넘기지 않는다.
3. 건강정보를 로그에 노출하지 않는다.
4. 치과의사 확인이 필요한 사안을 AI 단독 결론으로 표시하지 않는다.
