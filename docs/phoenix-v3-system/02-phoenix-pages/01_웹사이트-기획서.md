# phoenix pages 웹사이트 기획서

## 1. 정의

`phoenix pages`는 상세 페이지 생성 웹이다.

현재 구축된 `phoenix detail page`를 V3 시스템의 첫 번째 웹사이트로 발전시킨다.

## 2. 주 컨트롤 봇

```text
Design Bot
```

Design Bot은 상세페이지 이미지, 브랜드 비주얼, 섹션 구성, 카피 방향을 다루기 때문에 `phoenix pages`를 컨트롤하는 주 봇으로 적합하다.

## 3. 주요 유저

1. 마스터
2. 수강생 유저
3. 상품 판매자
4. 상세페이지 제작자
5. 광고/마케팅 담당자

## 4. 핵심 기능

1. 이미지/PDF 업로드
2. 판매 채널 선택
3. 생성 페이지 선택
4. 출력 비율 선택
5. 상세페이지 생성
6. 페이지 추가 생성
7. 개별 페이지 편집
8. AI 멘트 생성
9. 결과 ZIP 다운로드
10. V1 이후 Cloud 저장
11. V1.5 이후 크레딧 차감
12. V3 이후 Design Bot 자동 조작

## 5. 결과물

1. 상세페이지 이미지
2. 섹션별 이미지
3. ZIP 파일
4. 프로젝트 데이터
5. Cloud 저장 URL

## 6. V3 자동화 흐름

```text
Genesis Bot 또는 마스터
→ Design Bot에게 상세페이지 생성 요청
→ Design Bot이 phoenix pages 접속
→ 로그인 또는 API 키 설정
→ 파일 업로드
→ 생성 조건 선택
→ 결과 생성
→ ZIP 다운로드 또는 Cloud 저장
→ Telegram으로 결과 전달
```

## 7. 우선순위

`phoenix pages`는 이미 구축되어 있으므로 V3 시스템의 기준 웹사이트로 사용한다.

우선순위:

1. 현재 V0 기능 안정화
2. V1 Cloud 저장
3. V1.5 크레딧
4. V2 봇 자동화 준비
5. V3 Design Bot 제어
