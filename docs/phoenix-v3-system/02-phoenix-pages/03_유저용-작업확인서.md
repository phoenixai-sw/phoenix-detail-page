# phoenix pages 유저용 작업확인서

## 1. 목적

이 문서는 `phoenix pages`가 상세 페이지 생성 웹으로 정상 작동하는지 확인하기 위한 문서다.

## 2. 접속 확인

1. 사이트에 접속한다.
2. 메인 페이지에서 입장 버튼을 누른다.
3. 작업 화면으로 이동하는지 확인한다.

성공 기준:

```text
phoenix pages 작업 화면이 열린다.
```

## 3. API 키 설정 확인

1. `API 키 설정`을 연다.
2. OpenAI 또는 Google API 키를 입력한다.
3. 설정 상태가 바뀌는지 확인한다.

성공 기준:

```text
미설정 상태가 설정됨 상태로 바뀐다.
```

## 4. 상세페이지 생성 확인

1. 이미지 또는 PDF를 업로드한다.
2. 생성 페이지 수를 선택한다.
3. 출력 비율을 선택한다.
4. 요청 문구를 입력한다.
5. 생성 버튼을 누른다.

성공 기준:

```text
상세페이지 이미지가 생성된다.
```

## 5. 페이지 추가 생성 확인

1. 결과 마지막 페이지 뒤의 `페이지 추가 생성` 영역을 확인한다.
2. 1장, 2장, 3장 추가 버튼 중 하나를 누른다.
3. 새 페이지가 마지막 뒤에 추가되는지 확인한다.

## 6. ZIP 다운로드 확인

1. `결과 다운로드` 버튼을 누른다.
2. ZIP 파일 하나가 다운로드되는지 확인한다.
3. ZIP 안에 이미지가 들어 있는지 확인한다.

## 7. 봇 자동화 대비 확인

유저는 직접 볼 필요는 없지만, 코덱스 또는 운영자는 주요 버튼이 Design Bot에게 인식 가능한지 확인한다.

확인 대상:

1. 업로드
2. 생성 버튼
3. 다운로드 버튼
4. 결과 갤러리
5. Cloud 저장 버튼
