# phoenix reports 웹사이트 기획서

## 1. 정의

`phoenix reports`는 리서치/보고서 생성 웹이다.

시장조사, 경쟁사 분석, 정부지원사업 분석, 사업계획서 초안, 교육용 리포트를 생성한다.

## 2. 주 컨트롤 봇

```text
Power Bot
```

Power Bot은 리서치, 근거 정리, 보고서 구성, 분석 업무에 적합하다.

## 3. 핵심 기능

1. 보고서 주제 입력
2. 보고서 유형 선택
3. 참고자료 업로드
4. 리서치 범위 선택
5. 목차 생성
6. 본문 생성
7. 표/체크리스트 생성
8. 근거 링크 정리
9. PDF/DOCX 다운로드
10. Telegram 결과 전달

## 4. 추천 메뉴

1. 시장조사 보고서
2. 경쟁사 분석
3. 정부지원사업 분석
4. 사업계획서 초안
5. 강의 리서치 자료
6. 요약 보고서

## 5. 결과물

1. PDF
2. DOCX
3. 요약표
4. 근거 링크 목록
5. 발표용 요약본

## 6. 주의점

리서치 보고서는 최신 정보와 근거가 중요하다.

따라서 웹검색, 출처, 날짜, 근거 표시 원칙이 필요하다.
