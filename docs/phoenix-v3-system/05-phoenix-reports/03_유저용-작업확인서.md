# phoenix reports 유저용 작업확인서

## 1. 목적

`phoenix reports`가 리서치/보고서 생성 웹으로 정상 작동하는지 확인한다.

## 2. 보고서 생성 확인

1. 보고서 주제를 입력한다.
2. 보고서 유형을 선택한다.
3. 목차 생성을 누른다.

성공 기준:

```text
보고서 목차가 생성된다.
```

## 3. 본문 생성 확인

1. 목차를 확인한다.
2. 본문 생성을 누른다.
3. 본문과 표가 생성되는지 확인한다.

## 4. 근거 확인

1. 출처 목록이 표시되는지 확인한다.
2. 링크가 열리는지 확인한다.
3. 날짜가 표시되는지 확인한다.

## 5. 다운로드 확인

1. PDF 다운로드를 누른다.
2. DOCX 다운로드가 있으면 함께 확인한다.

## 6. Power Bot 자동화 확인

1. Telegram에서 Power Bot에게 보고서 생성을 요청한다.
2. Power Bot이 주제와 유형을 확인한다.
3. Power Bot이 phoenix reports를 조작한다.
4. 결과 보고서 파일을 전달받는다.
