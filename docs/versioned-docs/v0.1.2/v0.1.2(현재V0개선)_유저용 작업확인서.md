# v0.1.2 유저용 작업확인서

이 문서는 v0.1.2 상태의 phoenix detail page를 화면에서 확인하기 위한 체크리스트이다.

## 1. 접속 확인

1. 사이트에 접속한다.
2. 메인 화면에서 `phoenix detail page 입장` 버튼을 확인한다.
3. 버튼을 눌러 `제작실` 또는 `/studio` 화면으로 이동되는지 확인한다.

## 2. API 키 설정 확인

1. 왼쪽 또는 상단의 `API 키 설정` 버튼을 누른다.
2. OpenAI Image 2.0 API 키 입력칸을 확인한다.
3. Google Nano Banana 2 API 키 입력칸을 확인한다.
4. 저장 버튼을 눌렀을 때 설정 완료 메시지가 표시되는지 확인한다.

## 3. 제작실 확인

1. `원본 자료 가져오기` 영역이 보이는지 확인한다.
2. 이미지 또는 PDF 업로드가 가능한지 확인한다.
3. 제작 요청 메모 입력칸이 보이는지 확인한다.

## 4. 생성 페이지 확인

`생성 페이지` 메뉴에서 다음 옵션이 보이는지 확인한다.

- `1장(히어로)`
- `4장(심플)`
- `8장(상세)`

각 버튼을 눌렀을 때 선택 상태가 바뀌는지 확인한다.

## 5. 출력 비율 확인

`출력 비율` 메뉴에서 다음 옵션이 보이는지 확인한다.

- `9:16(상세)`
- `1:1(정사각)`
- `4:5(피드)`

각 옵션에는 용도 설명이 함께 표시되어야 한다.

## 6. 사용처 선택 확인

다음 옵션이 보이는지 확인한다.

- `스마트스토어`
- `쿠팡`
- `맞춤형 웹/앱`

도움말 버튼을 눌렀을 때 사용처별 설명이 표시되는지 확인한다.

## 7. 이미지 생성 확인

1. 원본 자료를 업로드한다.
2. API 키를 입력한다.
3. 생성 페이지와 출력 비율을 선택한다.
4. `이미지 제작 시작` 버튼을 누른다.
5. 생성 진행 화면이 표시되는지 확인한다.
6. 완료 후 `결과함`에 이미지가 표시되는지 확인한다.

## 8. 결과함 확인

결과 화면에서 다음 버튼을 확인한다.

- `결과 저장`
- `히어로 다시 생성`
- `결과 다운로드`
- 각 이미지의 `이미지 다운로드`
- 각 이미지의 `이 이미지 편집`

## 9. 페이지 추가 생성 확인

결과 마지막 카드 뒤에 `페이지 추가 생성` 카드가 보이는지 확인한다.

표시 내용:

- `현재 n장 / 최대 12장`
- `페이지 추가 생성`
- 원본 자료가 남아 있으면 더 정확하다는 TIP 안내
- 가능한 추가 버튼

현재 장수별 버튼 표시:

- 현재 8장: `1장 추가하기`, `2장 추가하기`, `3장 추가하기`
- 현재 10장: `1장 추가하기`, `2장 추가하기`
- 현재 11장: `1장 추가하기`
- 현재 12장: 추가 버튼 없음

추가 생성 버튼을 눌렀을 때 `제작실`로 이동하지 않고 `결과함`에 머무는지 확인한다.

## 10. 개별 편집 확인

1. 이미지 카드 아래의 `부분 편집 메모`에 수정 요청을 입력한다.
2. 편집 엔진을 선택한다.
3. `이 이미지 편집` 버튼을 누른다.
4. 수정된 이미지가 같은 카드에 반영되는지 확인한다.
5. 이전 이미지와 수정 이미지를 전환할 수 있는지 확인한다.

## 11. AI 멘트 생성 확인

빠른 수정 항목 옆의 `AI 멘트 생성` 버튼을 누른다.

확인할 항목:

- OpenAI API 키가 없을 때 안내 메시지가 표시되는지
- API 키가 있을 때 편집 메모가 자동으로 추가되는지

## 12. ZIP 다운로드 확인

1. 결과가 생성된 상태에서 `결과 다운로드`를 누른다.
2. 이미지가 여러 개 따로 내려받아지는 것이 아니라 ZIP 파일 하나로 다운로드되는지 확인한다.

## 13. 최근 제작 작업 확인

1. `결과 저장`을 누른다.
2. 홈보드로 이동한다.
3. `최근 제작 작업` 목록에 저장된 작업이 표시되는지 확인한다.
4. 제목 옆 연필 버튼으로 제목 수정이 가능한지 확인한다.
5. 체크 버튼으로 저장되는지 확인한다.
6. X 버튼으로 취소되는지 확인한다.
7. 휴지통 버튼으로 삭제가 가능한지 확인한다.

## 14. 현재 버전 판정

위 항목들이 모두 통과하면 현재 서비스는 v0.1.2 기준으로 정상 운영 가능한 상태로 본다.
