# phoenix detail page 사용 설명서

phoenix detail page는 기존 상세페이지 이미지나 PDF를 업로드해 AI로 상세페이지 이미지를 다시 생성하고, 결과를 저장, 편집, 다운로드할 수 있는 제작 도구입니다.

## 1. 시작하기

1. 사이트에 접속합니다.
2. 메인 화면에서 `phoenix detail page 입장` 버튼을 누릅니다.
3. 왼쪽 메뉴에서 `제작실`로 이동합니다.

## 2. API 키 설정

이미지 생성 전 `API 키 설정`에서 사용할 API 키를 입력합니다.

- `OpenAI Image 2.0 API 키`: OpenAI 이미지 생성/편집에 사용
- `Google Nano Banana 2 API 키`: Google 이미지 생성에 사용

입력한 키는 사용자의 브라우저에 저장됩니다.

## 3. 원본 자료 업로드

`원본 자료 가져오기` 영역에 기존 상세페이지 이미지 또는 PDF를 업로드합니다.

권장 자료:

- 기존 상세페이지 캡처 이미지
- 제품 패키지 이미지
- 상세페이지 PDF
- 리뷰, 인증, 구성, 혜택이 포함된 자료

업로드한 원본 자료가 현재 작업 화면에 남아 있을수록 추가 생성 정확도가 높아집니다.

## 4. 생성 옵션 설정

### 생성 페이지

- `1장(히어로)`: 첫 화면 테스트용
- `4장(심플)`: 짧은 상세페이지 또는 빠른 초안용
- `8장(상세)`: 기본 상세페이지 구성용

### 출력 비율

- `9:16(상세)`: 상세페이지 생성 기본값
- `1:1(정사각)`: 썸네일, 카드뉴스, 보조 이미지용
- `4:5(피드)`: 광고, SNS 피드 소재용

### 사용처 선택

- `스마트스토어`: 스캔이 쉬운 상품 정보 중심
- `쿠팡`: 빠른 구매 판단과 배송/후기 중심
- `맞춤형 웹/앱`: 브랜드, 서비스, 랜딩 페이지 흐름 중심

## 5. 이미지 생성

1. 원본 자료를 업로드합니다.
2. 생성 페이지, 출력 비율, 사용처를 선택합니다.
3. 제작 요청 메모를 입력합니다.
4. `이미지 제작 시작` 버튼을 누릅니다.

생성 중에는 진행 상태와 예상 대기 시간이 표시됩니다.

## 6. 결과 확인

생성이 완료되면 `결과함`에서 이미지를 확인할 수 있습니다.

결과 화면에서 가능한 작업:

- `결과 저장`: 현재 결과를 브라우저 저장소에 저장
- `결과 다운로드`: 생성된 이미지를 ZIP 파일 1개로 다운로드
- `이미지 다운로드`: 개별 이미지만 다운로드
- `이 이미지 편집`: 선택한 섹션만 부분 편집

## 7. 페이지 추가 생성

결과 마지막 페이지 뒤에 `페이지 추가 생성` 카드가 표시됩니다.

추가 가능한 버튼:

- `1장 추가하기`
- `2장 추가하기`
- `3장 추가하기`

최대 생성량은 12장입니다.

예시:

- 현재 8장: 1장, 2장, 3장 추가 가능
- 현재 10장: 1장, 2장 추가 가능
- 현재 11장: 1장 추가 가능
- 현재 12장: 추가 버튼 없음

정확도 안내:

처음 업로드한 이미지나 PDF가 현재 작업 화면에 남아 있을 때 제품 정보, 구성, 문구를 더 안정적으로 이어받습니다.

## 8. 개별 이미지 편집

각 결과 이미지 아래에서 부분 편집을 할 수 있습니다.

사용 방법:

1. 편집할 이미지 아래의 `부분 편집 메모`에 원하는 수정 내용을 입력합니다.
2. 필요하면 빠른 수정 버튼을 누릅니다.
3. `이 이미지 편집` 버튼을 누릅니다.

빠른 수정 버튼:

- 카피 강화
- 디자인 강화
- CTA 강화
- 모바일 첫화면
- 리뷰/UGC 신뢰
- 불안 제거
- 중복 레이아웃 줄이기
- 안전 표현

## 9. AI 멘트 생성

빠른 수정 버튼 옆의 `AI 멘트 생성`을 누르면 편집 요청 문구를 AI가 자동으로 만들어줍니다.

이 기능은 `API 키 설정`에 입력한 OpenAI API 키를 사용합니다.

## 10. 최근 제작 작업

홈보드의 `최근 제작 작업`에서 저장된 결과를 다시 열 수 있습니다.

가능한 작업:

- 제목 클릭: 결과 열기
- 연필 버튼: 제목 수정
- 체크 버튼: 제목 저장
- X 버튼: 제목 수정 취소
- 휴지통 버튼: 작업 삭제

## 11. 다운로드

`결과 다운로드` 버튼을 누르면 생성된 전체 이미지가 ZIP 파일 하나로 다운로드됩니다.

개별 이미지만 받고 싶으면 각 이미지 카드의 `이미지 다운로드` 버튼을 사용합니다.

## 12. 모델 이미지 규칙

모델이나 인물 이미지가 너무 반복 노출되지 않도록 생성 규칙이 적용되어 있습니다.

- 홀수 페이지: 모델/인물 1명 이하 허용
- 짝수 페이지: 제품컷, 아이콘, 표, 카드, FAQ, 근거 자료 중심

## 13. 저장 방식 안내

현재 기본 저장은 사용자의 브라우저 저장소를 사용합니다.

주의:

- 같은 브라우저에서는 저장된 작업을 다시 열 수 있습니다.
- 다른 PC나 다른 브라우저에서는 저장 내용이 보이지 않을 수 있습니다.
- 브라우저 데이터 삭제 시 저장된 작업이 사라질 수 있습니다.

## 14. 추천 사용 흐름

1. 원본 상세페이지 이미지 또는 PDF 업로드
2. `8장(상세)`과 `9:16(상세)` 선택
3. `스마트스토어`, `쿠팡`, `맞춤형 웹/앱` 중 사용처 선택
4. 이미지 제작 시작
5. 결과 확인
6. 부족한 부분은 `1장 추가하기`, `2장 추가하기`, `3장 추가하기`로 보강
7. 필요한 섹션만 개별 편집
8. 결과 저장
9. ZIP 파일로 다운로드
