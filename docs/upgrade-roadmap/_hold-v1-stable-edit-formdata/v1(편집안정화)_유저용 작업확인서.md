# V1 유저용 작업확인서

## 확인 목적

수강생은 개별 이미지 편집 기능이 큰 이미지에서도 안정적으로 동작하는지 확인한다.

## 확인 순서

1. 사이트에 접속한다.

2. API 키를 입력한다.

3. 기존 상세페이지 이미지 또는 PDF를 업로드한다.

4. 1장을 생성한다.

5. 결과 카드에서 `카피 강화` 또는 `디자인 강화` 버튼을 누른다.

6. `이 이미지 편집`을 누른다.

7. 수정된 이미지가 새 버전으로 추가되는지 확인한다.

8. 여러 섹션을 만든 뒤 S2, S3, S4도 같은 방식으로 편집한다.

## 성공 기준

1. `이미지 데이터가 너무 커서` 오류가 나오지 않는다.
2. 편집된 이미지가 기존 이미지 히스토리에 추가된다.
3. 이전 이미지와 수정 이미지를 넘겨볼 수 있다.
4. 마음에 드는 결과를 다운로드할 수 있다.

## 실패 시 유저 점검

1. API 키가 올바른지 확인한다.
2. 같은 섹션을 다시 한 번 편집해본다.
3. 업로드 원본이 지나치게 긴 세로 이미지라면 나눠서 업로드한다.
4. 문제가 반복되면 오류 메시지 화면을 캡처해 Codex에게 전달한다.
