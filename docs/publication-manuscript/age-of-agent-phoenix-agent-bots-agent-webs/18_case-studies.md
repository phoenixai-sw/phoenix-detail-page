이 문서는 원고에 넣을 수 있는 실제 사례형 설명을 제공한다.

## 사례: 상세페이지 생성 수업

사용 agent web: `phoenix pages`

담당 봇: `Design Bot`

상황: 수강생이 제품 자료를 올리고 상세페이지를 생성한다.

### 사건 흐름

1. 마스터가 작업 목적을 말한다.
2. Genesis Bot이 작업을 task로 나눈다.
3. Design Bot이 `phoenix pages`에 접속한다.
4. 필요한 입력값을 확인한다.
5. 생성 또는 분석을 실행한다.
6. 결과물을 outputs에 저장한다.
7. Telegram 전송 여부를 묻는다.
