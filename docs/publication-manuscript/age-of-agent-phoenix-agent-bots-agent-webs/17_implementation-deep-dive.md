# 구축과 운영 딥다이브

## 1. 이 문서의 목적

이 문서는 출판 원고의 실전 장을 더 깊게 만들기 위한 운영 설명 자료다.

각 섹션은 입문자가 실제 화면을 보며 따라 할 수 있도록 목표, 설명, 절차, 실패 대응으로 구성한다.

## GitHub 운영 딥다이브

### 핵심 요약

1. 저장소는 프로젝트 파일이 있는 원격 폴더로 설명한다.
2. clone은 원격 폴더를 내 컴퓨터로 가져오는 행위로 설명한다.
3. GitHub Desktop과 ZIP 다운로드의 차이를 입문자에게 알려준다.
4. 실제 운영에서는 clone을 권장하고, 단순 확인은 ZIP도 가능하다고 설명한다.

### 입문자 설명
