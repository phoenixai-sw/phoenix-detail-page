# 장별 확장 원고 초안

## 1. 이 문서의 목적

이 문서는 각 장을 실제 출판 원고로 확장할 때 사용할 문단 초안과 장면 자료를 제공한다.

집필 모델은 이 문서를 그대로 붙여 넣는 것이 아니라, 장별 서사와 실습 설명을 만드는 원천 재료로 사용한다.

## 1장 확장 초안: 에이전트 시대가 온다

### 장의 핵심

AI가 답변 도구에서 실행 도구로 바뀌는 흐름

### 독자가 떠올려야 할 장면

사람이 버튼을 누르던 과정을 봇이 일부 맡는 변화

### 원고 확장 문단

#### 확장 문단 1

에이전트 시대가 온다을 설명할 때 가장 중요한 것은 독자가 기술을 추상적인 구호로 받아들이지 않게 하는 것이다. 이 장에서는 AI가 답변 도구에서 실행 도구로 바뀌는 흐름을 실제 업무 장면으로 풀어야 한다.

예를 들어 사람이 버튼을 누르던 과정을 봇이 일부 맡는 변화을 보여주면 독자는 Phoenix 시스템이 왜 필요한지 더 쉽게 이해한다. 단순히 새로운 도구가 생겼다는 말보다, 사람이 반복하던 절차가 어떻게 봇과 웹의 협업 구조로 바뀌는지를 보여주는 편이 좋다.

이때 원고는 과장하지 않는다. 봇이 모든 것을 알아서 해결한다고 쓰지 않는다. 봇은 정해진 구조 안에서 안정적으로 움직일 때 강력해진다. 그래서 작업 지시, 상태 표시, 결과 저장, 승인 정책 같은 운영 장치가 중요하다.
